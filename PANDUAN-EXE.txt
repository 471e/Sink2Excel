PANDUAN PAKAI EXE
=================

Isi paket:
- Sinkronisasi-Excel-Gambar.exe
- Converter-HEIC-ke-JPG.exe
- PANDUAN-EXE.txt

1. Sinkronisasi-Excel-Gambar.exe
   Fungsi:
   - Sinkronisasi file gambar ke kolom Excel
   - Mendukung Foto KTP, Foto Rumah, dan Foto Meter Listrik
   - Mendukung kolom opsional Foto Lainnya jika foldernya dipilih manual

   Cara pakai:
   - Jalankan Sinkronisasi-Excel-Gambar.exe
   - Klik Browse Excel
   - Pilih sheet jika ingin proses satu sheet
   - Klik Isi Folder Default atau pilih folder manual
   - Jika sheet punya kolom Foto Lainnya, pilih foldernya manual
   - Klik Sinkronisasi Sheet Terpilih atau Sinkronisasi Semua Sheet Valid
   - Simpan file hasil

   Output:
   - File Excel hasil sinkronisasi
   - Sheet Log Sinkronisasi
   - Sheet Detail Tidak Cocok jika ada mismatch

2. Converter-HEIC-ke-JPG.exe
   Fungsi:
   - Convert file HEIC ke JPG
   - Termasuk file .jpg/.jpeg yang ternyata isi aslinya HEIC

   Cara pakai:
   - Jalankan Converter-HEIC-ke-JPG.exe
   - Klik Gunakan Folder Default atau Tambah Folder
   - Pilih Folder Output
   - Klik Scan Kandidat HEIC
   - Klik Convert ke JPG

   Catatan:
   - File asli tidak diubah
   - Hasil JPG disimpan ke folder output terpisah

3. Catatan penting
   - Jika SmartScreen Windows muncul, klik More info lalu Run anyway bila Anda yakin sumber file ini benar
   - Sebaiknya simpan hasil kerja di folder yang Anda punya hak tulis
   - Untuk hasil terbaik, pakai folder gambar lokal yang lengkap
   - Paket rilis ini sudah lolos uji workflow otomatis untuk sinkronisasi Excel dan converter HEIC
