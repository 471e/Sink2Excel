# GitHub Release Checklist v1.0.1

Ikuti langkah ini saat membuat GitHub Release untuk `v1.0.1`.

## Sebelum Publish

- Pastikan tag `v1.0.1` sudah ada di GitHub.
- Pastikan branch `main` sudah sinkron dengan `origin/main`.
- Pastikan paket `Sinkronisasi-Excel-Tools-v1.0.1.zip` sudah tersedia di lokal.
- Siapkan isi deskripsi dari `GITHUB_RELEASE_PUBLISH_v1.0.1.md`.

## Langkah Di GitHub

1. Buka repository `https://github.com/471e/Sink2Excel`.
2. Klik menu `Releases`.
3. Klik `Draft a new release`.
4. Pada `Choose a tag`, pilih `v1.0.1`.
5. Pada `Release title`, isi `Sink2Excel v1.0.1`.
6. Salin isi bagian `Description` dari `GITHUB_RELEASE_PUBLISH_v1.0.1.md`.
7. Upload file `Sinkronisasi-Excel-Tools-v1.0.1.zip` sebagai asset release.
8. Pastikan target branch/tag yang tampil tetap `v1.0.1`.
9. Klik `Publish release`.

## File Pendukung

- `GITHUB_RELEASE_v1.0.1.md`
- `GITHUB_RELEASE_SHORT_v1.0.1.md`
- `GITHUB_RELEASE_PUBLISH_v1.0.1.md`

## Catatan

- Jangan pindahkan tag `v1.0.1` ke commit dokumentasi terbaru.
- Tag `v1.0.1` tetap menunjuk ke commit rilis yang sudah divalidasi.
